# great_meadow_project
This repository was created to synthesize, process, and analyze freshwater wetland data collected in Great Meadow and Gilmore Meadow in Acadia National Park as part of the Great Meadow Project. This repository also prepares data for visualization in the [great_meadow_apps](https://github.com/Nportales/great_meadow_apps) repository. Components of this repository are adapted from the [wetlandACAD R package](https://github.com/katemmiller/wetlandACAD).
